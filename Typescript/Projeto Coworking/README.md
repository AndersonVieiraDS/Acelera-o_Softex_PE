# CoworkTech - Sistema de Reservas de Salas

O **CoworkTech** é um sistema de reserva de salas para espaços de coworking. Ele oferece funcionalidades para gerenciar usuários, espaços disponíveis e reservas.

## Pré-requisitos

- [Node.js](https://nodejs.org/) e [npm](https://www.npmjs.com/) instalados no seu ambiente.
